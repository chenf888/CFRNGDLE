# CFRNGDLE

一个 11 位随机数字生成器 + 徽章收集游戏，原型为 [RNGdle](https://www.rngdle.com)。

## 玩法

点击 **Generate** 生成一个 11 位随机数，从右到左逐位闪烁揭示。根据数字的数学性质和模式获得徽章，每个徽章有 TP 分值。

### 徽章系统

约 140+ 个徽章，分 8 大类：

| 分类 | 示例 |
|------|------|
| 位数徽章 | 十一位数 → 一位数 |
| 数学性质 | 质数、完全数、斐波那契、平方数… |
| 数字模式 | 升序/降序、回文、三连~十连、山峰/山谷… |
| 网络文化梗 | 69、42、233、520、911、114514… |
| 连续序列 | 012~901、1234~7890… |
| 数字密集 | ≥5~9 个相同数字 |
| π/e/φ 常数 | 连续匹配 3~11 位 |
| 奇趣模式 | 费曼点、键盘形、之字形… |

9 种稀有度：平庸 → 普通 → 罕见 → 稀有 → 史诗 → 传说 → 神话 → 超越 → 终结

## 技术

纯前端，无依赖 — HTML/CSS/JS 单文件组合。数据通过 localStorage 持久化。
