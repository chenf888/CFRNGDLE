# CFRNGDLE

一个 11 位随机数字生成器 + 徽章收集游戏，原型为 [RNGdle](https://www.rngdle.com)。

## 玩法

点击 **Generate** 生成一个 11 位随机数，从右到左逐位闪烁揭示。根据数字的数学性质和模式获得徽章，每个徽章有 TP 分值。

徽章列表按稀有度排序（最稀有关在前），默认显示前 20 个，可展开查看全部。本轮新获得的徽章置顶标注 ✨ 新获得。

## 徽章系统

约 **240 个**徽章，分 15 大类：

| 分类 | 数量 | 示例 |
|------|------|------|
| 位数徽章 | 11 | 十一位数 → 一位数（TP 1~100 亿）|
| 数学性质 | 30 | 质数、半质数、盈数、完全数、平方数、立方数、2 的幂、斐波那契… |
| 数字模式 | 25 | 升序/降序、回文、三连~十连、山峰/山谷… |
| 网络文化梗 | 52 | 69、42、233、520、911、114514、66686、CSGO 炸弹、Lost 数字… |
| 连续序列 | 35 | 012~901、1234~7890、一二三四五… |
| 数字密集 | 20 | ≥5~10 个相同数字 |
| π/e/φ 常数 | 15 | 连续匹配 3~11 位 |
| 奇趣模式 | 16 | 费曼点、之字形、键盘形、两两成十… |
| 恰好一个 X | 10 | 恰好出现一次 0~9 |
| 邻居/对子 | 8 | 邻居、乌龟爬、三明治、对称对、悬崖、高原… |
| 幂次数 | 5 | 3 的幂、5 的幂、三角数、阶乘、卡塔兰数 |
| 化学元素 | 10 | 氢(1)~氖(0)，以原子序数命名 |
| 哈沙德/求和 | 10 | 哈沙德数、莫兰数、快乐数、亏数、和为 11~55… |
| 数学猜想 | 12 | 梅森素数、哥德巴赫、拉马努金 1729、卡普雷卡 6174… |
| 只含特定数字 | 10 | 全质数字、十全十美、只有 0 和 1、奇偶平衡… |
| 黄金分割 & 更多 | 18 | φ 连续匹配、√2/√3/√5 前缀、ABAB/ABCABC 模式… |

9 种稀有度：**平庸 → 普通 → 罕见 → 稀有 → 史诗 → 传说 → 神话 → 超越 → 终结**

## 数据持久化

所有已获得徽章和总 TP 自动保存在浏览器 localStorage，刷新不丢失。

## 技术

纯前端，无依赖 — HTML / CSS / 原生 JS 四模块组合。

```
HYPERNGDLE.html   → 入口页面
style.css          → 全部样式
badge-utils.js     → 工具函数库
badge-defs.js      → 240 个徽章定义
badges.js          → 徽章管理模块
app.js             → 主应用逻辑
```
